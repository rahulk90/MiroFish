HOW TO USE THESE FILES
================================================================================

SCENARIO 1: I'm new to the project, where do I start?
  1. Read: TRANSLATION_PROJECT_SUMMARY.txt (15 min)
  2. Read: QUICK_REFERENCE_GUIDE.txt (10 min)
  3. Choose a component (Step3 or Step5 - no backend dependencies)
  4. Read: Component-specific translation list
  5. Start translating!

SCENARIO 2: I'm ready to translate Step3Simulation.vue
  1. Open: Step3Simulation_Chinese_Translation_List.txt
  2. Read: Summary statistics section
  3. Review: Template section (UI elements first)
  4. Review: Script section (messages and logs)
  5. Use: QUICK_REFERENCE_GUIDE.txt for terminology
  6. Translate using the line numbers as guides
  7. Use: Verification checklist (in QUICK_REFERENCE_GUIDE.txt)

SCENARIO 3: I'm translating Step4Report.vue
  1. Stop! Read: CRITICAL WARNING section in summary
  2. Contact: Backend team about regex patterns
  3. Wait for: Backend update confirmation
  4. Then: Open Step4Report_Chinese_Translation_List.txt
  5. Focus on: CRITICAL regex patterns first
  6. Then: HIGH priority UI text
  7. Test: Regex patterns against updated backend output

SCENARIO 4: I'm translating Step5Interaction.vue
  1. Similar to Step3 workflow (no backend dependencies)
  2. Open: Step5Interaction_Chinese_Translation_List.txt
  3. Note: Tool descriptions are detailed (MEDIUM priority)
  4. Translate: Following the line-by-line guide
  5. Verify: Using QUICK_REFERENCE_GUIDE.txt checklist

SCENARIO 5: I need a terminology glossary
  1. Reference: TRANSLATION_PROJECT_SUMMARY.txt
  2. Section: "APPENDIX: TRANSLATION GLOSSARY (DRAFT)"
  3. Reference: QUICK_REFERENCE_GUIDE.txt
  4. Section: "COMMON TRANSLATIONS FOR ALL FILES"

SCENARIO 6: I need to understand backend dependencies
  1. Read: TRANSLATION_PROJECT_SUMMARY.txt
  2. Section: "CRITICAL FINDINGS & WARNINGS"
  3. Reference: QUICK_REFERENCE_GUIDE.txt
  4. Section: "REGEX PATTERN MAPPING (For Step4Report Backend Update)"
  5. Action: Contact backend team with this information

SCENARIO 7: I'm verifying translations are complete
  1. Use: Verification checklist in QUICK_REFERENCE_GUIDE.txt
  2. Run: grep command to find remaining Chinese characters
  3. Compare: Against translation list items
  4. Test: Component functionality
  5. Review: Log messages appear in English

SUPPORT & QUESTIONS
================================================================================

Q: Where do I find the English translations to use?
A: In the component-specific translation lists. Each file has the original
   Chinese and suggested English translation.

Q: What if I find a better translation?
A: Great! Make a note and we can update the glossary for consistency.

Q: Can I start with Step4Report.vue?
A: NOT RECOMMENDED. It's blocked waiting for backend updates. Start with
   Step3 or Step5 instead.

Q: What if a translation seems wrong?
A: Check the context section in the translation list. If still unclear,
   review the code surrounding that string.

Q: Should I translate comments?
A: Yes, but they're lower priority (LOW). Focus on user-facing text first.

Q: How do I know when I'm done translating?
A: Use the grep command to verify no Chinese characters remain, then check
   the verification checklist in QUICK_REFERENCE_GUIDE.txt.

Q: What about technical terms I'm not sure about?
A: Consult the glossary in QUICK_REFERENCE_GUIDE.txt or ask the team.
